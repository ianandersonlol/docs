[Homebrew](https://brew.sh/) is an essential package manager for macOS that makes installing and managing software incredibly easy. Think of it as an app store for command-line tools and applications.

## Installation

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

After installation, add Homebrew to your PATH. The installer will print commands specific to your system; the conditional version below works for both Apple Silicon and Intel Macs:

```bash
if [[ "$(uname -m)" == "arm64" ]]; then
    BREW_PREFIX="/opt/homebrew"
else
    BREW_PREFIX="/usr/local"
fi

echo "eval \"\$(${BREW_PREFIX}/bin/brew shellenv)\"" >> ~/.zprofile
eval "$(${BREW_PREFIX}/bin/brew shellenv)"
brew --version
```

Homebrew's official default prefix is `/opt/homebrew` on Apple Silicon and `/usr/local` on Intel macOS.

## Why Homebrew is Essential for Bioinformatics

- **Dependency Management**: Automatically handles complex dependencies between tools
- **System Tooling**: Great for stable utilities you use across many projects
- **Large Package Catalog**: Many common bioinformatics tools are available directly as core formulas
- **Easy Updates**: Keep all your tools up to date with simple commands
- **Binary Distribution**: Pre-compiled packages mean faster installation than building from source

## What Should Go in Homebrew?

Use Homebrew for system-level tools and GUI applications:

| Good fit for Homebrew | Better elsewhere |
|-----------------------|------------------|
| `git`, `wget`, `jq`, `htop`, `tree` | Project-specific bioinformatics toolchains |
| VS Code, RStudio, IGV | Reproducible pipeline environments |
| General developer utilities | R package libraries for a specific analysis |

For reproducible analysis environments, use [Miniforge + Mamba](/misc-bioinfo/mac-setup/miniforge). For R analysis projects, use R/RStudio with `renv` when you need package snapshots.

## Understanding Formulas vs Casks

Homebrew has two main types of packages:

### Formulas (Command-Line Tools)

**Formulas** are command-line tools and libraries installed with `brew install`:

```bash
# Examples of formulas
brew install samtools
brew install python
brew install wget
brew install git
```

Formulas are typically:
- Command-line programs
- Libraries and dependencies
- Open-source software
- Installed to `/opt/homebrew/bin/` (Apple Silicon) or `/usr/local/bin/` (Intel)

### Casks (GUI Applications)

**Casks** are GUI applications installed with `brew install --cask`:

```bash
# Examples of casks
brew install --cask rstudio
brew install --cask visual-studio-code
brew install --cask github
brew install --cask igv
```

Casks are typically:
- Applications with graphical interfaces
- Installed to `/Applications/`
- Sometimes proprietary software
- .app bundles, .pkg installers, or .dmg files

## Homebrew Tutorial: Common Operations

### Basic Installation

```bash
# Install a formula (command-line tool)
brew install samtools

# Install a cask (GUI application)
brew install --cask rstudio

# Install another bioinformatics formula
brew install blast
```

### Searching for Packages

```bash
# Search all packages
brew search blast

# Search only formulas
brew search --formula blast

# Search only casks
brew search --cask genome

# Search by description text
brew search --desc alignment
```

### Getting Information

```bash
# Get info about a package
brew info samtools

# List installed packages
brew list

# List only formulas
brew list --formula

# List only casks
brew list --cask
```

## Updating and Upgrading

### Update Homebrew Itself

```bash
# Update Homebrew's package definitions (do this regularly!)
brew update
```

This updates Homebrew's local database of available packages and versions, but doesn't upgrade your installed software.

### Upgrade Installed Packages

```bash
# See what packages can be upgraded
brew outdated

# Upgrade ALL packages (formulas and casks)
brew upgrade

# Upgrade specific formula
brew upgrade samtools

# Upgrade specific cask
brew upgrade --cask rstudio

# Upgrade only formulas
brew upgrade --formula

# Upgrade only casks
brew upgrade --cask
```

### Maintenance Commands

```bash
# Clean up old versions and cache
brew cleanup

# See what cleanup would remove (dry run)
brew cleanup --dry-run

# Check system for potential problems
brew doctor

# Get help
brew help
```

## Bioinformatics-Specific Tips

### Prefer Core Formulas First

Most bioinformatics tools you need on macOS are available directly in Homebrew core formulas and casks.

```bash
# Search before adding taps
brew search samtools
brew search blast
brew search bwa
```

If a tool is missing from Homebrew, install it via bioconda in a dedicated environment instead of adding old/unmaintained taps.

### Common Bioinformatics Installations

```bash
# Quick local bioinformatics utilities
brew install samtools bcftools bedtools
brew install blast bwa

# GUI applications (casks)
brew install --cask igv
brew install --cask jalview
brew install --cask figtree
```

<Note>
  Homebrew bioinformatics formulas are useful for quick local work. For a project or pipeline you need to reproduce later, install the same tools in a mamba environment and save an `environment.yml`.
</Note>

### Best Practices

```bash
# Weekly maintenance routine
brew update
brew outdated

# Upgrade intentionally when you are ready
brew upgrade
brew cleanup

# Before starting a new project, check your installation
brew update && brew doctor

# Check what's installed
brew list | grep bio
```
