VS Code has become the go-to editor for bioinformatics work thanks to its extensive extension ecosystem, integrated terminal, and remote development support. It handles everything from Python scripts to Jupyter notebooks to quick FASTA file previews.

```bash
brew install --cask visual-studio-code
```

## Enable the `code` Command

The extension commands below require the `code` command to be available in Terminal:

1. Open VS Code
2. Press `Cmd+Shift+P`
3. Search for `Shell Command: Install 'code' command in PATH`
4. Run it, then restart Terminal

Verify:

```bash
code --version
code .
```

## Essential Extensions for Bioinformatics

These are some great extensions to make VS Code work even better, but totally optional!

```bash
# Install from command line
code --install-extension ms-python.python
code --install-extension ms-toolsai.jupyter
code --install-extension REditorSupport.r
code --install-extension yzhang.markdown-all-in-one
code --install-extension redhat.vscode-yaml
code --install-extension ms-vscode-remote.remote-ssh
code --install-extension github.vscode-github-actions
```

### Key Extensions

| Extension | Purpose |
|-----------|---------|
| **Python** | Full Python support with debugging and IntelliSense |
| **Jupyter** | Work with Jupyter notebooks directly in VS Code |
| **R** | R language support and debugging |
| **Markdown All in One** | Essential for writing documentation and papers |
| **YAML** | For configuration files and workflow definitions |
| **Remote - SSH** | Edit files directly on HPC clusters or lab servers |
| **GitHub Actions** | Inspect workflow files and CI status |

If you plan to use R inside VS Code, install the R language server from R:

```r
install.packages("languageserver")
```

## VS Code Tips for Bioinformatics

1. **Integrated Terminal**: Use `` Ctrl+` `` to open terminal without leaving the editor
2. **Remote Development**: Edit files on remote servers and HPC clusters
3. **Git Integration**: Built-in version control with visual diff tools
4. **Multi-file Editing**: Work with large projects spanning multiple languages

## Recommended Settings for Bioinformatics

Open settings (JSON) with `Cmd+Shift+P` → "Preferences: Open Settings (JSON)" and add:

```json
{
  "editor.fontSize": 14,
  "editor.wordWrap": "on",
  "files.associations": {
    "*.fasta": "plaintext",
    "*.fa": "plaintext",
    "*.fastq": "plaintext",
    "*.fq": "plaintext",
    "*.vcf": "plaintext",
    "*.bed": "plaintext",
    "*.gff": "plaintext",
    "*.gtf": "plaintext"
  },
  "editor.renderWhitespace": "boundary",
  "files.trimTrailingWhitespace": true
}
```
