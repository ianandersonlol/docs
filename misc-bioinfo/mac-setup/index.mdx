Setting up a new Mac for bioinformatics work can feel overwhelming with the sheer number of tools. This guide will walk you through installing and configuring the essential tools you need to get started with computational biology workflows. Whether you're analyzing genomic data, building phylogenetic trees, or developing bioinformatics pipelines, this setup will provide a solid foundation.

## Quick Start Order

For the best experience, follow the guides in this order:

1. **Prerequisites** - Install Xcode Command Line Tools first (required for everything else)
2. **Homebrew** - Install the package manager
3. **CLI Tools** - Get essential utilities
4. **GitHub** - Set up version control
5. **VS Code** - Configure your editor
6. **Miniforge + Mamba** - Set up conda/mamba for bioinformatics packages
7. **R & RStudio** - Install R ecosystem
8. **Shell Customization** - Add helpful aliases and functions
9. **Terminal Theme** - (Optional) Beautify your terminal
10. **Troubleshooting** - Come back here when something breaks

## Final Smoke Test

After finishing the core setup, open a new terminal window and run:

```bash
# System development tools
xcode-select -p
git --version

# Homebrew
brew --version
brew doctor

# Core command-line tools
htop --version
tree --version
jq --version

# GitHub and VS Code
gh auth status
code --version

# Conda/mamba
conda --version
mamba --version
conda env list

# R
R --version
```

Most commands should print a version number. `brew doctor` may print warnings, but it should not report a broken installation. `gh auth status` should say you are logged in after you finish the GitHub setup.

<Tip>
  If one of these commands says `command not found`, go to [Troubleshooting](/misc-bioinfo/mac-setup/troubleshooting), especially the PATH section.
</Tip>
