Setting up a new Mac for bioinformatics work can feel overwhelming with the sheer number of tools. This guide will walk you through installing and configuring the essential tools you need to get started with computational biology workflows. Whether you're analyzing genomic data, building phylogenetic trees, or developing bioinformatics pipelines, this setup will provide a solid foundation.

## Quick Start Order

For the best experience, follow the guides in this order:

1. **Prerequisites** - Install Xcode Command Line Tools first (required for everything else)
2. **Homebrew** - Install the package manager
3. **CLI Tools** - Get essential utilities
4. **GitHub** - Set up version control
5. **VS Code** - Configure your editor
6. **Terminal Theme** - (Optional) Beautify your terminal
7. **Mambaforge** - Set up conda/mamba for bioinformatics packages
8. **R & RStudio** - Install R ecosystem
9. **Shell Customization** - Add helpful aliases and functions
