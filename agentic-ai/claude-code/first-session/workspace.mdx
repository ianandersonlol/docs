Before you do anything useful, Claude Code needs to know *where* to work. For a non-coder, this is the most unfamiliar idea in the whole guide — so we'll take it slow.

**What this page will cover:**
- The mental model of a "folder" / "project" and why everything happens inside one
- Creating a dedicated workspace folder for a task
- Opening that folder in the Claude Code desktop app
- Putting your input files (spreadsheets, PDFs) where Claude can see them
- What Claude can and can't touch outside the folder
