This section comes in three tracks, written for different starting points. They don't build on each other — start with whichever one matches the work in front of you.

<CardGroup cols={2}>
  <Card title="Agentic AI for Bioinformatics" icon="dna" href="/agentic-ai/bioinformatics">
    **For scientists and researchers.** What "agentic" really means, an opinionated tour of the tool landscape, and hands-on guides to skills, slash commands, and subagents for your pipelines. Assumes you're already comfortable in a terminal.
  </Card>
  <Card title="Working with Spreadsheets" icon="table" href="/agentic-ai/spreadsheets">
    **For analysts, finance teams, and operators.** When to use Claude for Excel, Claude chat, Claude Cowork, and Claude Code for spreadsheet work, with practical habits for managing workbook content safely.
  </Card>
  <Card title="Claude Code for Beginners" icon="graduation-cap" href="/agentic-ai/claude-code">
    **For business, finance, and ops folks who've never written code.** Install Claude Code, run your first real projects, and build a working financial forecasting model — all in plain English, no coding required.
  </Card>
</CardGroup>

## Which one is me?

| If you… | Go to |
|---|---|
| Work in research and want agents in your computational workflows | [Agentic AI for Bioinformatics](/agentic-ai/bioinformatics) |
| Spend most of your time in Excel and want Claude to help with models, exports, formulas, and reporting | [Working with Spreadsheets](/agentic-ai/spreadsheets) |
| Come from PE, finance, consulting, or operations and have never thought of yourself as "technical" | [Claude Code for Beginners](/agentic-ai/claude-code) |
| Just want to understand what an AI agent *is* before doing anything | [What is Agentic AI?](/agentic-ai/what-is-agentic-ai) |

<Note>
  These tracks overlap on purpose. The spreadsheet track is the most Excel-specific, the beginner track is the broader non-technical on-ramp, and the bioinformatics track goes deeper and more technical.
</Note>
